# ComfyUI-FunPack Dev Branch

This is the development branch of ComfyUI-FunPack.

Features on this branch may be unfinished, unstable, broken, or changed without notice. If you want a stable version of this node set, do not use the `dev` branch. Use the current version from the `main` branch instead.

Issues with nodes from the `dev` branch will not be reviewed, and reports about them will be ignored. Feedback is accepted only for issues with the current version of nodes present on the `main` branch.
