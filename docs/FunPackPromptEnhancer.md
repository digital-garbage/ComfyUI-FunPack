# FunPack Prompt Enhancer

This node uses an LLM to rewrite a simple user prompt into a richer video description.

## Parameters

**user_prompt**: Short prompt or concept you want to expand.

**system_prompt**: Instruction telling the LLM how to structure the enhanced prompt.

**model_path_type**: Choose between a local `.safetensors` LLM file and a HuggingFace pretrained model.

**model_path**: HuggingFace repository path used when loading a pretrained model.

**llm_safetensors_file**: Local `.safetensors` model file from your ComfyUI `models/clip` folder.

**top_p, top_k, temperature, max_new_tokens, repetition_penalty**: Text generation settings that control variety and output length.

## Outputs

**enhanced_prompt**: The rewritten prompt generated by the LLM.

## Purpose

Use this node when you want a rough idea turned into a more detailed video prompt before passing it into the rest of your workflow.
